# 🎥 Karpagam Memories - Video Dashboard

[![Website Status](https://img.shields.io/badge/Status-Live-brightgreen)](https://college-memories-git-main-jayanivaasan.vercel.app/)

🌐 **Live Demo**: https://college-memories-git-main-jayanivaasan.vercel.app/

> **"Preserving memories, creating futures."**

Karpagam Memories is a sleek, modern video platform designed for the students and faculty of **Karpagam College of Engineering (KCE)**. It serves as a centralized hub to relive campus events, explore departmental achievements, and manage personal video favorites.

## ✨ Key Features

- 📱 **Responsive Dashboard**: A premium, mobile-first design with smooth scroll-reveal animations.
- 📂 **Departmental Categorization**: Browse academic videos and projects organized by branch (CSE, EEE, MECH, etc.).
- 📅 **Event Highlights**: Quick access to recent campus celebrations, symposiums, and cultural festivals.
- 🔒 **User Authentication**: Secure login and registration system for personalized experiences.
- 💾 **My Vault**: A dedicated section for users to save and revisit their favorite memories.
- 🎬 **Integrated Player**: Custom video playback interface optimized for a seamless viewing experience.

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge).

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/JAYANIVAASAN/Karpagam-Memories-Video.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd Karpagam-Memories-Video
    ```
3.  **Open the project**:
    -   Simply open `index.html` in your browser.
    -   *Recommended*: Use the **Live Server** extension in VS Code for the best experience.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure and modern layouts.
- **CSS3**: Premium styling using Glassmorphism, CSS Variables, and advanced Flexbox/Grid layouts.
- **JavaScript (ES6+)**: Dynamic content injection, slider logic, and interaction management.
- **SVG Icons**: Lightweight and scalable vector graphics for a sharp UI.
